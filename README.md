# Depression-Symptoms-Analysis-and-global-trend-analysis-using-Exploratory-Data-Analysis-EDA-
This project focuses on analyzing depression symptom frequency using Python-based Exploratory Data Analysis (EDA). The dataset was explored to understand relationships between different depression symptom frequencies such as Nearly every day, More than half the days, Several days, and Not at all. Using data visualization techniques    
🔍 Important Insights
1. Strong Positive Correlation Between Frequent Symptoms

People reporting symptoms:

Nearly every day
More than half the days
Several days

all show very high positive correlations (0.90 – 0.99).

➡️ This means individuals experiencing one symptom frequently are highly likely to experience other symptoms frequently as well.

2. “Not at all” Has Strong Negative Correlation

The “Not at all” category shows strong negative correlation with all other symptom frequencies.

Example:

With “Several days” = -0.99
With “Total Frequency” = -1.00

➡️ This indicates that when depression symptoms increase, responses for “Not at all” decrease significantly.

3. Total Frequency Closely Matches Symptom Severity

The Total Frequency column has extremely high positive correlations with:

Several days → 0.99
More than half the days → 0.97
Nearly every day → 0.95

➡️ This suggests total symptom frequency is a strong indicator of overall depression severity.

4. Clear Pattern in Mental Health Trends

The heatmap visually shows:

Red colors → strong positive relationships
Blue colors → strong negative relationships

➡️ This helps quickly identify mental health symptom trends and behavioral patterns.
# Depression Symptoms EDA Project

## 📌 Overview
This project performs Exploratory Data Analysis (EDA) on depression symptom data to identify patterns, relationships, and symptom frequency trends.

The project uses Python data analysis libraries to visualize correlations between different depression symptom categories.

---

## 🎯 Objectives
- Analyze depression symptom frequency
- Identify relationships between symptoms
- Visualize correlations using heatmaps
- Generate meaningful healthcare insights

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Key Visualizations
- Correlation Heatmap of Depression Symptoms
- Frequency Distribution Analysis
- Comparative Symptom Analysis

---

## 🔍 Key Insights
- Strong positive correlation exists between frequent depression symptoms.
- “Not at all” responses are negatively correlated with other symptom frequencies.
- Total symptom frequency strongly indicates depression severity.
- Heatmaps help identify mental health trends effectively.
